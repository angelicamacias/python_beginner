## Characteristics

If you want circles insted of a line use: 

## Shape
```
"o"
```
For example: 
```
plt.plot([1,2,3,4], [1,4,5,6], "o")
```
## Color 
Or we can assign the line to a variable and then say the shape:

```
graph = plt.plot([1,2,3,4], [1,4,5,6], "o")
graph.setp(lines, color="#ff5566") = 
```
